# Населени места в България

Пълна база данни на всички населени места в България с пощенски кодове, ЕКАТТЕ кодове, телефонни кодове, GPS координати, области, общини и кметства.

## Структура на данните

Файлът `bulgaria_places_data.json` е организиран йерархично по **област → община → ЕКАТТЕ код**:

```json
{
  "Име на област": {
    "Име на община": {
      "ЕКАТТЕ код": {
        "name": "Име на населеното място",
        "type": "Тип на населеното място",
        "town_hall": "Кметство",
        "phone_code": "Стационарен телефонен код",
        "latitude": 42.123456,
        "longitude": 23.456789,
        "post_codes": [
          "4000",
          "4001"
        ]
      }
    }
  }
}
```

### Описание на полетата

| Поле | Тип | Описание |
|------|-----|----------|
| `name` | `string` | Официалното наименование на населеното място |
| `type` | `string` | Тип на населеното място (виж таблицата по-долу) |
| `town_hall` | `string` | Наименование на кметството, към което принадлежи |
| `phone_code` | `string` | Стационарен телефонен код (напр. `"032"` за Пловдив) |
| `latitude` | `number` | GPS географска ширина (десетична степен) |
| `longitude` | `number` | GPS географска дължина (десетична степен) |
| `post_codes` | `string[]` | Масив с пощенски кодове (едно място може да има повече от един) |

### Типове населени места

| Код | Тип |
|-----|-----|
| `гр.` | Град |
| `с.` | Село |
| `к.к.` | Курортен комплекс |
| `ман.` | Манастир |
